OSU Mappack Downloader
=======================
1. Place all your beatmap links (one per line) in links.txt
2. Run osu_mappack_downloader.exe
3. Log in using your osu! account
4. Choose a folder name
5. All .osz maps will be downloaded and zipped for you

Requires internet connection ofc 
Only works with valid osu! client credentials

Enjoy!
